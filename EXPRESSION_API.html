Evaluate

Signature: math.evaluate(expr: string | string[], scope?: object | Map<string, any>) → any | any[] Scope parameter: plain object or Map<string, any> or custom with get(key), set(key,value), has(key), keys(). Assignment in expressions writes directly to scope. Object is wrapped in ObjectWrappingMap. Custom functions scope wrapped in PartitionedMap.

Signature: math.compile(expr: string | string[]) → CompiledExpression CompiledExpression.evaluate(scope?: object | Map<string, any>) → any | any[] Evaluate compiled code against optional scope. Scope types same as math.evaluate.
