# README_EXAMPLES

Goal
Document usage examples and a compact conversion table in README.md so users can quickly use the library.

Behavior
- Show how to import named exports and call toRoman and fromRoman in plain examples.
- Provide an example converting 1994 to MCMXCIV and converting MCMXCIV back to 1994.
- Include a compact conversion table for common values.

Acceptance Criteria
- README contains examples demonstrating both toRoman and fromRoman usage
- README contains the conversion table mapping common values: 1=I, 4=IV, 5=V, 9=IX, 10=X, 40=XL, 50=L, 90=XC, 100=C, 400=CD, 500=D, 900=CM, 1000=M
