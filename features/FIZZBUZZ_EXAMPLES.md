# FIZZBUZZ_EXAMPLES

Overview
Provide runnable examples in the examples/ directory that demonstrate common uses of the library and CLI.

Specification
- Add examples/node/basic.js that imports the library and prints fizzBuzz(15) when run with node.
- Add examples/cli/README.md that documents CLI commands and expected output.
- Examples must be runnable without modification on the repository Node version.

Files to change
- examples/node/basic.js (new)
- examples/cli/README.md (new)
- README.md (link to examples)

Acceptance Criteria
1. examples/node/basic.js runs with node and prints the expected 15-line sequence.
2. examples/cli/README.md contains exact commands to run the CLI and the expected output.
3. README links to the examples directory.
