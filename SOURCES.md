# Node.js File System (fs) Documentation
## https://nodejs.org/api/fs.html
The definitive reference for Node.js filesystem operations, covering synchronous and asynchronous APIs, file and directory streaming, buffer pooling, highWaterMark tuning, and advanced fs/promises and AbortSignal-sensitive methods. Essential for reliably writing JSON, NDJSON, and SVG/PNG payloads to disk or stdout, and for efficient bulk data persistence in Node.js v20. Published 2023; Core runtime docs, authoritative.
