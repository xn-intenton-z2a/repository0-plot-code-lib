# Sources

Reference material and documentation sources for this project.

- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math  — MDN: Math object reference
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Function  — MDN: Function constructor (dynamic function creation, scope, security)
- https://developer.mozilla.org/en-US/docs/Web/SVG/Element/polyline  — MDN: SVG polyline element (points attribute, stroke/fill)
- https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/viewBox  — MDN: SVG viewBox attribute (min-x min-y width height)
- https://datatracker.ietf.org/doc/html/rfc4180  — RFC 4180: Common Format and MIME Type for CSV Files
- https://nodejs.org/api/fs.html  — Node.js fs module (readFile, writeFile, streams)
- https://developer.mozilla.org/en-US/docs/Web/API/Canvas_API/Tutorial/Basic_usage  — MDN: Canvas API basic usage (drawing, exporting PNG)
- https://sharp.pixelplumbing.com/  — sharp: high performance Node.js image processing (SVG -> PNG conversion)

The maintain-library workflow will process these into `library/` documents.
