SVG PATH

Overview

The element in SVG defines a shape by using a series of commands and parameters. It is the generic element to create complex shapes and can be used to draw lines, curves, and arcs.

Attributes

  • d

    • Description: Defines the shape of the path.
    • Value type: string
    • Default value: ''
    • Animatable: yes

  • pathLength

    • Description: Specifies the total length of the path in user units.
    • Value type: number
    • Default value: none
    • Animatable: yes

Code Example

A typical usage in an HTML document:

html, body, svg { height: 100%; }

Browser Compatibility

  • The element is widely supported across devices and browser versions since January 2020.
  • Both the 'd' and 'pathLength' attributes are animatable, enhancing interactive and dynamic SVG graphics.
